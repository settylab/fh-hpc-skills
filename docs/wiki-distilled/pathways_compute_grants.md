# Scientific Computing Resources for Grant Writers

Source: https://sciwiki.fredhutch.org/scicomputing/compute_grants/

## Key Facts

- SciComp maintains a document "Fred-Hutch-Computational-Resource-Description-for-Grant-Writers.docx" with current, frequently updated specs including non-public details
- Request the latest version by emailing SciComp directly
- Fred Hutch computational resources are partially funded through NIH S10 instrumentation grants

## Required Citation for Publications

When using Fred Hutch computational resources, include this attribution:

> "Fred Hutch Scientific Computing, NIH grants S10-OD-020069 and S10-OD-028685"

## How to Get the Grant Description

Email SciComp (scicomp@fredhutch.org) to request the latest version of the grant-writer resource description document. The document contains details about:

- Cluster hardware specifications (CPU, GPU, memory, storage)
- Storage infrastructure (Fast File, Temp, Economy Cloud)
- Software and tools available
- Support services
- Network and security infrastructure

## Common Pitfalls

- Using outdated hardware specs in grant applications (always request the latest document)
- Forgetting to include the NIH S10 citation in publications
- Trying to find detailed specs on the wiki instead of requesting the document from SciComp
