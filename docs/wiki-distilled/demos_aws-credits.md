# AWS Cloud Credits

Source: https://sciwiki.fredhutch.org/compdemos/aws-credits/

## Virtual Credits

CIT BizOps converts AWS credits into virtual credits by spending them immediately and issuing monthly credit memos. Unlike standard credits limited to single accounts, virtual credits can fund both test and production environments simultaneously.

## Sandbox Accounts

Request requirements: project name/description, account owner details, timeline, HutchNet IDs with admin/user designations, special requirements.

Features: Separate credentials from lab accounts, mandatory MFA, broad service permissions, data transfer with economy-cloud S3 buckets.

Restrictions: Cannot directly manage Route53, KMS keys, AWS Config, logging/monitoring, user groups, VPC networking, or tags. All require CLD assistance.
