# Compute Overview (Distilled)

Source: https://sciwiki.fredhutch.org/scicomputing/compute_overview/

## Key Facts

- Landing page for the LARGE SCALE COMPUTING section of the Fred Hutch SciComp wiki.
- The batch system used at Fred Hutch is Slurm.
- Interactive computing options: RStudio Server (web-based R IDE) and Jupyter Lab (web-based Python/R notebooks).
- Sub-topics: Quick Start, Technologies/Platforms, Computing Environments, Scientific Software, Job Management, Parallel Computing, Cloud Computing, GPU Computing.
- Community-curated Slurm examples live in the FredHutch/slurm-examples GitHub repository.

## Commands & Examples

None on this page (it is a navigation hub).

## Common Pitfalls

- New users may jump into job submission without understanding the Slurm fundamentals covered in the sub-pages.
- The overview itself contains minimal technical content; all details are in sub-pages.
