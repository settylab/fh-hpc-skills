## Shell Scripting

### Purpose

Automates repetitive tasks across multiple files (e.g., renaming sequencing run files, adjusting microscopy image contrast).

### Available Shells

- **Bash** (Bourne Again SHell): Standard on most Unix-based systems including Ubuntu
- **zsh**: Default on macOS, based on Bash
- **PowerShell/Command Prompt**: Windows environments

### Benefits

1. No additional software installation required
2. Built-in accessibility across platforms
