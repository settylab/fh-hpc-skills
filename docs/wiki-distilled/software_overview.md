# Software Overview at Fred Hutch

Source: https://sciwiki.fredhutch.org/scicomputing/software_overview/

## Common Programming Languages

- **R and RStudio** -- widely used among statisticians, software development environment with IDE
- **Python** -- accessible for learning, strong community modules
- **Linux/Unix/Shell (BASH)** -- foundation for cluster work

## Decision Factors for Language Selection

- What colleagues in your work group use
- Tools most common in your field
- Career goals and resume value
- Personal syntax preferences

## Computing Resources

Code can run on local computers or shared clusters via Large Scale Computing resources (Rhino/Gizmo).

## Proprietary Software Supported

SAS, SPSS, Stata, Matlab, Tableau

## Workflow Managers

- **Actively supported**: Nextflow and WDL workflows
- **Also mentioned**: Snakemake, Cromwell, Toil, CWL

## Other Languages

Julia, Go/GoLang, Perl

## Support Channels

- Slack communities: `#workflow-managers`, `#tableau-user-comm`, fhdata workspace
- GitHub repository for contributions
- Resource Library with demos (including Nextflow)
- Email SciComp for software requests
