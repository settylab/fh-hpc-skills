# Code Examples and Templates at Fred Hutch

Source: https://sciwiki.fredhutch.org/scicomputing/software_examples/

## Coding Practices (8 Principles)

1. Separate raw and processed data storage
2. Organize source code and results with clear labeling
3. Include license and README files with project overview
4. Fully document code with additional documentation folders
5. Reuse existing code and packages
6. Automate code to minimize manual errors
7. Use inline comments and meaningful variable names for readability
8. Include Help documentation for Python packages/modules
