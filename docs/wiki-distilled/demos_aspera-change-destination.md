# Aspera Client Download Path Configuration

Source: https://sciwiki.fredhutch.org/compdemos/aspera-change-destination/

## Changing Default Download Location

1. Right-click Aspera icon (macOS/Linux) or use Hamburger Button (Windows)
2. Open "Transfers" preferences tab
3. Choose: fixed path in "save to" field, or "always prompt" for flexibility
4. Apply changes before downloading

"Always prompt" is recommended for greater flexibility across sessions.
