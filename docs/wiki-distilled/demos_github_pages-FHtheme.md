# Fred Hutch GitHub Pages Theme

Source: https://sciwiki.fredhutch.org/compdemos/github_pages-FHtheme/

## Overview

Custom GitHub Pages theme based on Minimal Mistakes Jekyll theme with Fred Hutch branding.

## Configuration

In `config.yml`, this setting must remain unchanged:
```yaml
minimal_mistakes_skin: "contrast"
```
Changing to any other skin removes institutional styling.
