# Access Overview (Distilled)

Source: https://sciwiki.fredhutch.org/scicomputing/access_overview/

## Key Facts

- This is the landing page for the ACCESS & CREDENTIALS section of the Fred Hutch SciComp wiki.
- Three sub-topics: Credentials, Methods, Permissions.
- Credentials: HutchNet ID, GitHub, AWS, Rhino/Gizmo access.
- Methods: SSH connections, SSH clients, SSH config, VPN.
- Permissions: Data storage permission organization and troubleshooting.

## Commands & Examples

None on this page (it is a navigation hub).

## Common Pitfalls

- New users may not realize they need separate credential setup steps for different systems (HutchNet, GitHub org, AWS).
- The overview page itself is minimal; the real content is in the sub-pages.
